# adaptive-pulse

心拍数トリガーでフェーズが切り替わるアダプティブ・インターバルトレーニングの Wear OS アプリ (Pixel Watch 向け・個人利用)。

固定時間ではなく心拍で運動を制御する: 高強度フェーズで心拍が上限閾値を超えたら振動で知らせて回復フェーズへ、回復フェーズで下限閾値を下回ったら振動で高強度フェーズへ戻る。目標サイクル数に達するか、疲労の蓄積 (上限到達が速くなりすぎ) を検知したら自動で終了する。運動中の操作は不要で、通知はすべて振動で行う。

> ⚠️ 本アプリは医療機器ではありません。運動とその強度設定は自己責任で行ってください。
