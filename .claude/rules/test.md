# テスト

「動くことを保証する」ことを最優先に、それなりの精度で書く。

- **unit test**: ステートマシンの分岐 (閾値遷移・タイムアウト終了・疲労ブレーキ・終了判定) を、時計と心拍ソースの fake で網羅する。JVM 上で完結し、エミュレータ・実機なしで通ること。
- instrumented test / エミュレータ上のテストは必要になったときだけ追加する。
- 実測の心拍データをテストに使わない (合成データを使う。`secret.md` 参照)。
- push 前に `./gradlew build` (test 含む) が必ず通ること (PreToolUse hook と CI が強制する)。
